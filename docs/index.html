Open Cloud Day México

Conferencia gratuita presentada por Microsoft y Red Hat enfocada en desarrollo de aplicaciones y gestión de infraestructura de cómputo en nube híbrida.

10 de diciembre de 2019, Ciudad de México.

Regístrate

Open Cloud Day es un día de conferencias técnicas gratuitas presentado por Microsoft y Red Hat con el objetivo de avanzar el conocimiento de las prácticas y tecnologías para construir aplicaciones e infraestructura en la nube con alto desempeño e interoperabilidad.

Dirigido a

Desarrolladores de aplicaciones web y móviles, arquitectos de software, CTOs, ingenieros de datos.

Temas

Esta edición del Open Cloud Day estará principalmente enfocada en cómo y por qué utilizar software de Red Hat en Azure.

  • Introducción a OpenShift.
  • Laboratorios prácticos para desplegar aplicaciones en Azure Red Hat OpenShift.
  • Gestión de autenticación y autorización.
  • Migración de bases de datos a la nube.
