Horario Actividad

9:30 - 11:00 Construye una nube empresarial híbrida con Red Hat y Azure, por Héctor Valtierra.

Ventajas del modelo híbrido de cómputo en la nube basado en contenedores; oferta conjunta de Red Hat y Microsoft en Azure.

11:30 - 12:30 Introducción a OpenShift, por Javier Árce.

Daremos un repaso de qué es OpenShift, su relación con Kubernetes, arquitectura y conceptos clave.

 Puntos clave para migrar bases de datos a la nube, por Rolando Cedillo

Conoce qué manejadores de base de datos open source son soportados en Azure, así como algunas estrategias y recomendaciones para lograr el desempeño, seguridad, flexibilidad y costo que buscas.
12:30 - 13:30 Hands-on lab: Creación y gestión de un cluster Azure Red Hat OpenShift, por Javier Árce.

En este laboratorio práctico aprenderás a crear y configurar los elementos básicos para construir un ambiente OpenShift en Azure.

 Usando contenedores en la nube, por Cristiam Espinoza

Conoce qué manejadores de base de datos open source son soportados en Azure, así como algunas estrategias y recomendaciones para lograr el desempeño, seguridad, flexibilidad y costo que buscas.

14:30 - 15:30 Hands-on lab: Despliegue y monitoreo de una app en Azure OpenShift, por Javier Árce.

En este laboratorio práctico desplegaremos los componentes de una aplicación (base de datos, backend y frontend) a nuestro ambiente de OpenShift en Azure.

 Seguridad en la nube con Red Hat y Active Directory, por Rogelio Oliva.

Conoce las estrategias y pasos para gestionar usuarios y autenticación, así como las opciones de cifrado y otros aspectos de seguridad al utilizar OpenShift en Azure.
15:30 - 16:00 (... continúa ...) Opciones de entrenamiento y certificación para profesionistas, por Rolando Cedillo.
